# 📋 WBS Manager

個人用のWBS（Work Breakdown Structure）運用管理アプリケーションです。

## 機能

- **WBSテーブル管理** - タスクの追加・編集・削除（インライン編集対応）
- **階層構造** - 親タスク配下にサブタスクを作成可能
- **進捗管理** - 未着手 / 進行中 / 完了 / 中断 / 取消 のステータス管理
- **日程管理** - 予定・実績の開始日・終了日を管理
- **ガントチャート** - 並行タスクの可視化（予定と実績を色分け表示）
- **フィルター** - ステータス・カテゴリでの絞り込み
- **データ永続化** - SQLite によるローカルDB保存

## 技術スタック

| 種別 | 技術 |
|------|------|
| フロントエンド | Vite + React + TypeScript |
| バックエンド | Node.js + Express + TypeScript |
| データベース | SQLite（better-sqlite3） |

## スクリプトを使った操作（推奨）

プロジェクトルートに以下のPowerShellスクリプトを用意しています。

| スクリプト | 説明 |
|-----------|------|
| `start.ps1` | バックエンド・フロントエンドを別ウィンドウで同時起動 |
| `stop.ps1` | 起動中のサービスをポート番号から特定して停止 |
| `status.ps1` | 起動状況の確認とヘルスチェック |

### 起動

```powershell
.\start.ps1
```

### 停止

```powershell
.\stop.ps1
```

### 起動状況の確認

```powershell
.\status.ps1
```

> **注意**: スクリプトの実行が拒否される場合は、以下のコマンドで実行ポリシーを変更してください。
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```

## サービスの起動方法

> **バックエンドとフロントエンドをそれぞれ別のターミナルで起動してください。**

### ① バックエンドAPIサーバーを起動（ターミナル1）

```bash
npm run server
```

起動成功時の表示:
```
🚀 WBS APIサーバーが起動しました: http://localhost:3001
```

### ② フロントエンド開発サーバーを起動（ターミナル2）

```bash
npm run dev
```

起動成功時の表示:
```
VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

### ③ ブラウザでアクセス

```
http://localhost:5173/
```

---

## 起動状況の確認方法

### バックエンドの確認

ブラウザまたはcurlでヘルスチェックにアクセス:

```bash
curl http://localhost:3001/api/health
```

正常時のレスポンス:
```json
{"status":"ok","timestamp":"2026-03-05T08:00:00.000Z"}
```

### ポートの使用状況確認（PowerShell）

```powershell
# バックエンド（3001番ポート）が起動しているか確認
netstat -ano | findstr :3001

# フロントエンド（5173番ポート）が起動しているか確認
netstat -ano | findstr :5173
```

出力があればサービスが起動中、出力がなければ停止中です。

## サービスの停止方法

各ターミナルで `Ctrl + C` を押すと停止します。

| サービス | ポート | 停止方法 |
|---------|--------|---------|
| バックエンドAPIサーバー | 3001 | ターミナル1で `Ctrl + C` |
| フロントエンド開発サーバー | 5173 | ターミナル2で `Ctrl + C` |

## 基本的な使い方

| 操作 | 方法 |
|------|------|
| タスク追加 | ツールバー「＋ タスク追加」ボタン |
| セル編集 | セルをクリック → 入力 → Enter またはフォーカスアウトで確定 |
| サブタスク追加 | 行右端の「+」ボタン |
| タスク削除 | 行右端の「×」ボタン（確認ダイアログあり） |
| ステータス変更 | 進捗列のドロップダウンを選択 |
| ガントチャート表示 | ツールバー右上のビュー切り替えボタン |
| フィルター | ツールバーのステータス / カテゴリドロップダウン |

## WBSの項目構成

| カラム | 説明 |
|--------|------|
| 項番 | 自動採番 |
| カテゴリ | タスクの大分類（企画、設計、開発など） |
| タスク | 作業項目（階層構造でサブタスクを追加可能） |
| 担当者 | 担当者名 |
| 進捗 | ステータス（未着手 / 進行中 / 完了 / 中断 / 取消） |
| 予定 開始 / 終了 | 予定期間 |
| 実績 開始 / 終了 | 実績期間 |
| 備考 | メモ・補足情報 |
